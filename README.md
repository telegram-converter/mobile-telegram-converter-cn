# MTCR 转换器  
**移动 Telegram → SESSION+JSON 或 TDATA**

> 现代、安全且快速的 Telegram 移动账号转换工具

将 **官方 Telegram (tgnet.dat)** 与 **Telegram X (td.binlog)** 文件转换为：

- **SESSION+JSON**（兼容 Telethon / Pyrogram）
- **TDATA**（Telegram Desktop / Telegram Prime 格式）

### ⚙️ 核心功能

| 功能 | 说明 |
|------|------|
| 输入格式 | 官方 Telegram (tgnet.dat) + Telegram X (td.binlog) |
| 输出格式 | SESSION+JSON（Telethon/Pyrogram）或 TDATA |
| 多线程 | 可设置任意数量的并发转换线程 |
| 会话行为 | 保留当前会话 或 完全重新登录 |
| 会话清理 | 选项：在转换期间终止所有其他活跃会话 |
| 连接调优 | 自定义重试次数 + 尝试间隔延迟 |
| 两步验证 | 自动从 TXT 文件导入或为所有账号统一输入一条验证码 |
| 设备模拟 | 灵活的 Android/iOS/桌面模拟（旧会话与新会话可单独设置） |
| 代理支持 | 每个线程唯一 HTTP/SOCKS5 代理（支持多行列表） |
| 结果排序 | 有效 / 无效 / 超时 / 两步验证错误 → 分离文件夹 |
| 源备份 | 原始文件在处理前自动复制 |
| 详细日志 | 实时进度与错误显示在「程序控制台日志」窗口 |
